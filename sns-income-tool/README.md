# SNS自動収益化ツール

X（Twitter）でバズ投稿をリサーチ → リライト → 自動投稿する収益化パイプライン。

## 仕組み

```
バズ投稿リサーチ → 6種のリライト → 収益化CTA追加 → ゴールデンタイム自動投稿
```

### 収益化の流れ
1. **リサーチ**: DuckDuckGoでバズっている投稿を自動収集（APIキー不要）
2. **リライト**: 6つのスタイルでオリジナルコンテンツに変換
3. **CTA追加**: アフィリエイト/リード獲得/エンゲージメント向上のCTAを自動付与
4. **自動投稿**: エンゲージメントが高い時間帯（7時/12時/18時/21時）に自動投稿
5. **分析**: 投稿パフォーマンスを記録・レポート表示

## セットアップ

```bash
cd sns-income-tool
pip install -r requirements.txt
cp .env.example .env
# .env にX APIキーを設定
```

### X API キーの取得
1. https://developer.twitter.com でアプリ作成
2. API Key, API Secret, Access Token, Access Token Secret を取得
3. `.env` に設定

## 使い方

### 1. バズ投稿をリサーチ
```bash
python main.py research "副業"
python main.py research "AI" -n 10
```

### 2. 1週間分の投稿を自動生成
```bash
python main.py generate 副業
python main.py generate AI --days 14 --posts-per-day 3 --cta affiliate
```

### 3. スケジュールをプレビュー
```bash
python main.py preview
```

### 4. 自動投稿を実行
```bash
python main.py run
```

### 5. パフォーマンスレポート
```bash
python main.py stats
```

### その他
```bash
python main.py niches    # 対応ジャンル一覧
python main.py styles    # リライトスタイル一覧
python main.py post "テスト投稿です"  # 即座に1件投稿
```

## リライトスタイル

| スタイル | 説明 | 例 |
|---------|------|-----|
| hook | 注目フック追加 | 【知らないと損】... |
| list | 箇条書き変換 | ①... ②... ③... |
| question | 問いかけ形式 | 【質問】...知ってましたか？ |
| story | 体験談形式 | これを知って人生変わった。... |
| reverse | 逆視点リライト | 逆の視点で言うと... |
| concise | 一文抽出 | 最もインパクトのある一文 |
